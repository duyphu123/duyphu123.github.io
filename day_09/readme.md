## khai báo biến

sử dụng từ khóa `let` , `const` để khai báo biến

### khai báo biến bth
```javascripst
// khai báo biến
let number;
// khai báo biến và khởi tạo giá trị
let age = 18
```
được sử dụng khai báo hằng số
```javascript
const numebr = 18
```
